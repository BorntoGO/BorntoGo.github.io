功能:接了一个人体红外感应传感器(23),探测到就亮灯(24),没什么技巧跟led一样一样的,

```
# -*- coding:utf-8 -*-
__author__ = 'Kyle Yuan'

import RPi.GPIO as GPIO
import time

GPIO.setmode(GPIO.BCM)
GPIO.setup(23, GPIO.IN)
GPIO.setup(24, GPIO.OUT)
while True:
    i = GPIO.input(23)
    if i == 1:
        GPIO.output(24, 1)
    else:
        GPIO.output(24, 0)
    time.sleep(2.5)
    #同类产品说明书说探测到后延时2.5秒,此处按此处理
```
